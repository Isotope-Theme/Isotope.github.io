Rofi

Automatic

To use the automatic installer, clone the repository, and run the installer script.

git clone https://github.com/Isotope-Theme/Rofi.git
cd Rofi
./install.sh

This will prompt the user to pick between the light and dark variants and a primary color, and either append the color scheme to the existing .Xresources, or creates a new one.

Manual

A second alternative is to clone the repository, and manualy copy or link the Xresources of your selected variant to your home directory as .Xresources.
