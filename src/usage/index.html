Usage

User Settings

The user should be able to set some settings, that will effect the end result of the color scheme. Primarily this includes a Primary and Secondary color. Using these colors should change the main and secondary colors of the theme. The user selected colors should be selected from one of the default color set colors.

There should always be two variants of the color scheme. One with a dark background, and one with a light background.

Default Colors

The background color should be either Grey 1 for light variants or Grey 10 for dark variants. Then the foreground should be the other color of those two. So if the background is Grey 10, then the foreground should be Grey 1.

The accent colors should be an assortment chosen from the default colors. Commonly go with Red, Purple, Blue, Cyan, Green, and Amber when the number of colors is limited. Such as for a terminal color scheme.
