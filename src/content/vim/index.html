Vim

Pathogen

If you use pathogen by tpope:

cd ~/.vim
git clone https://github.com/Isotope-Theme/vim bundle/isotope-theme

Vundle

If you use vundle:

Plugin 'Isotope-Theme/vim'
:PluginInstall

Vim-Plug

If you use vim-plug:

Plug 'Isotope-Theme/vim'
:PlugInstall

Manual

Copy the isotope.vim file into ~/.vim/colors

Enable the Theme

Add the following lines somewhere in your vimrc file:

syntax on
colorscheme isotope
