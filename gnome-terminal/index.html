Isotope

A dark theme for Gnome Termainal

Gnome Terminal

To install the gnome terminal themes for Isotope, run the following command. This command will download the installation script. Then it must be run. The installer will prompt you for either the light or dark theme.

wget https://raw.githubusercontent.com/Isotope-Theme/gnome-terminal/master/isotope.sh
bash isotope.sh
