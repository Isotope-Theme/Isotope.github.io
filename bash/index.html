Isotope

A dark theme for Bash

Bash

Install using Git

If you use git, you can clone the repo to your local .bashrc or .bash_profile

. 
$ git clone https://github.com/isotope-theme/bash.git

Using Bash-It

If you use Bash-It you can clone the bash it theme from the same directory. Then simply add the following line to your .bashrc or .bash_profile

export BASH_IT_THEME='$ISOTOPE_THEME'

Note: Replace $ISOTOPE_THEME with the directory path to the isotope theme
